

Load data into DynamoDB table using step functions

1) Create table in DynamoDB table
 - TransactionHistoryTable
 - TransactionId ( TransactionId )

2) Create step function to orchestrate the data pipeline
 - Add step definition
 - start execution
 - input to the execution




Step function definition
--------------------------------------
{
  "Comment": "Transaction Processor State Machine",
  "StartAt": "ProcessTransaction",
  "States": {
    "ProcessTransaction": {
      "Type": "Pass",
      "Next": "StoreHistory"
    },
    "StoreHistory": {
      "Type": "Task",
      "Resource": "arn:aws:states:::dynamodb:putItem",
      "Parameters": {
        "TableName": "TransactionHistoryTable",
        "Item": {
          "TransactionId": {
            "S.$": "$.TransactionId"
          }
        }
      },
      "Retry": [
        {
          "ErrorEquals": [
            "States.ALL"
          ],
          "IntervalSeconds": 1,
          "MaxAttempts": 3
        }
      ],
      "End": true,
      "ResultPath": "$.DynamoDB"
    }
  }
}
